Theory: Torque–Slip Characteristics
An induction motor develops torque through electromagnetic induction between the stator and rotor. The "slip" represents the difference between the speed of the magnetic field and the actual rotor speed. Understanding this curve is vital for controlling starting torque and motor efficiency.

Theory: Nyquist Stability Criterion
This criterion is a graphical method for determining the stability of a feedback control system. It uses the frequency response of the open-loop transfer function to predict how the closed-loop system will behave.

Q3. Nyquist Stability Criterion Analysis

Consider the following regarding the Nyquist Plot:

  1. Determines closed-loop stability using open-loop response.
  2. Stability depends on encirclements of (–1, 0).
  3. Applies only to minimum phase systems.
  4. Encirclements depend on number of open-loop RHP poles.
A partial view that misses the mathematical relationship involving Right Half Plane poles.
Correct choice. It correctly links stability to the open-loop response and pole-encirclement math.
Incorrect. Statement 3 is a common misconception; Nyquist is a universal tool for all phase types.

Final Answer: Option B

N = Z − P

N = Encirclements, Z = Closed-loop RHP poles, P = Open-loop RHP poles

✔ Statement 1: Uses open-loop G(s)H(s) to predict stability of the total system.

✔ Statement 2: The critical point for evaluating stability is (–1, 0).

❌ Statement 3: Nyquist is powerful because it applies to both minimum and non-minimum phase systems.

✔ Statement 4: Encirclement count is a direct function of the number of open-loop RHP poles.
