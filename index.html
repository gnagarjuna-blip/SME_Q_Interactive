Q1. Induction Motor Torque Characteristics

Question: Consider the following statements regarding torque–slip characteristics of an induction motor:

  1. Maximum torque is independent of rotor resistance.
  2. Maximum torque is proportional to square of applied voltage.
  3. At starting condition, slip is zero.
  4. Increasing rotor resistance shifts maximum torque towards higher slip.

Which of the above statements is/are correct?

A) 1 and 2 only
Option A is partially correct but incomplete because it ignores Statement 4, which is a fundamental property of slip-ring motors.
B) 1, 2 and 4 only
This is the correct choice. It includes all physical truths while excluding the false claim that slip is zero at starting.
C) 2 and 3 only
This is incorrect because Statement 3 is fundamentally wrong; slip is maximum (1) at start, not zero.
D) 1, 3 and 4 only
This is incorrect as it includes Statement 3, which misidentifies the starting slip value.
✅ Correct Answer: Option B

Theory: Torque-Slip Relation

The torque in an induction motor is generated by the interaction of the stator magnetic field and the induced rotor currents. The "slip" represents the relative difference between the synchronous speed and the actual rotor speed. At standstill, slip is 1; at synchronous speed, slip is 0.

Governing Equation:

Torque = (3 ÷ ωs) × (s E22 R2) ÷ (R22 + (s X2)2)

Maximum Torque Equation:

Tmax = (3 ÷ 2ωs) × (E22 ÷ X2)

Statement Analysis

  • Statement 1: Correct. Looking at the Tmax equation above, R2 is absent.
  • Statement 2: Correct. Torque is directly tied to E22, which is proportional to V2.
  • Statement 3: Incorrect. At start, speed is 0, so Slip = (Speedsync - 0) ÷ Speedsync = 1.
  • Statement 4: Correct. The slip at which max torque occurs is sm = R2 ÷ X2. Higher R2 means higher sm.

Q3. Nyquist Stability Criterion

Question: Consider the following statements regarding Nyquist plot:

  1. Nyquist plot determines closed-loop stability using open-loop response.
  2. Stability depends on encirclements of (–1, 0).
  3. Nyquist criterion applies only to minimum phase systems.
  4. Encirclements depend on number of open-loop RHP poles.
A) 1 and 2 only
Incomplete. It misses the crucial relationship between encirclements and open-loop poles (Statement 4).
B) 1, 2 and 4 only
Correct. It combines the methodology, the critical point, and the mathematical requirement for RHP poles.
C) 2 and 3 only
Incorrect. Statement 3 is false because Nyquist is famous for handling non-minimum phase systems.
✅ Correct Answer: Option B

Theory: Stability Analysis

The Nyquist Criterion is a frequency-domain method. It maps the Nyquist contour from the s-plane to the G(s)H(s) plane. The number of times the plot encircles the critical point (-1, 0) tells us if the closed-loop system will be stable.

The Stability Formula:

N = Z - P

Where N is clockwise encirclements, Z is closed-loop RHP poles, and P is open-loop RHP poles.

Statement Analysis

  • Statement 1: Correct. It allows us to predict closed-loop behavior without calculating closed-loop poles.
  • Statement 2: Correct. The point (-1, j0) represents the boundary of instability.
  • Statement 3: Incorrect. It works for minimum and non-minimum phase systems alike.
  • Statement 4: Correct. The formula N = Z - P directly utilizes the count of open-loop poles in the Right Half Plane.
